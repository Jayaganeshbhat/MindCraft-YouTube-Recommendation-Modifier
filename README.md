# MindCraft - AI Powered YouTube Learning Assistant

An AI-powered learning platform that transforms YouTube into a personalized learning environment.

Instead of recommending random entertainment videos, MindCraft generates structured learning paths based on a user's goals and replaces YouTube recommendations with educational content through a Chrome Extension.

## Features

- 🤖 AI-powered learning path generation using Gemini API
- 🎯 Goal-based YouTube recommendations
- 🔒 Google OAuth authentication
- 📚 Personalized learning dashboard
- 🧩 Chrome Extension with Focus Mode
- 📺 Replace YouTube homepage with curated learning videos
- 👤 User profile & learning progress
- 📦 Full-stack architecture
