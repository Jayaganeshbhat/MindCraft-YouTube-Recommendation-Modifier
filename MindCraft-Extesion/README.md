# Mindcraft Chrome Extension

A Chrome extension that connects with your Mindcraft learning platform.  
It helps you stay focused by replacing YouTube’s homepage with videos from your learning paths.

---

## Login

1. Click the Mindcraft icon in your Chrome toolbar.
2. If you’re not logged in, click **“Sign in with Google”**.
3. This opens your Mindcraft login (same as the web app).
4. Once signed in, reopen the popup to see your profile and learning paths.

---

## Focus Mode

1. Toggle **Focus Mode** ON in the popup.
2. Visit **YouTube** – the homepage will show only YouTube videos from your learning paths.
3. Toggle it OFF to restore the normal YouTube view.

---

## Learning Paths

- **Create Path** → Opens the web app to create a new learning path.
- **Dashboard** → Opens the full Mindcraft dashboard.
- **View Paths** → Click any path name to open it in the app.

---

## Developer Setup

1. Open Chrome and go to:
```chrome://extensions/```
2. Enable **Developer Mode** (top right).
3. Click **Load unpacked**.
4. Select the extension folder (where `manifest.json` is located).
5. Pin it to the toolbar for quick access.

---

That’s it! The extension is ready to help you learn smarter and stay focused.
